README -----------> New age of paint!<------------------------

How to install ?
1.Open 'paint(darkChecker)'
2.Run 'paint_2.0'
3.Add to bookmarks

How to Use ?
0-9: standart colors
e: paint all
r: clear all
u, i ,y: control panel
g: make/destroy markup
m, n ,b: tools
=/- : lineWidth
shift + "+/-": circles of 'b' tool
x: Palette!
Tap on color to add in 'rgb'
Tap on 'done!' to save color
o: past image (enter URL)
p: return image
'F11': full display
Enter: start! 
!--- IF TOOLS NOT WORKING, YOU MUST CHANGE  KEYBOARD  LAYOUT ---!
